# App de Livros e Mangás

## Nome

Vinícius Gabriel Inacio

## Descrição

Aplicativo mobile feito com Ionic + Vue 3.
Possui sistema de cadastro e login, além de uma página para recuperação de senha.

Após entrar, é possível visualizar uma lista de livros e mangás e ver seus detalhes.
Os dados são simulados por meio do mockData.ts

## Telas

* Login
* Cadastro
* Recuperação de senha
* Home (lista de livros e mangás)
* Detalhes

## Como rodar

```bash
npm install
ionic build
ionic serve
```
